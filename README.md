### Deku Daemon
#### How to build
_Project's root dir_\
`make -B start`\
`./start`
#### How to run
##### `--c [configuration file]`
* `DIR_ISP=""`
* `STD_NAME_REQUEST_FILE=""`
* `DIR_REQUEST_FILE=""`
* `DIR_SUCCESS=""`
* `DIR_SCRIPTS=""`
* `SOCKET_HOST=""`
* `SOCKET_PORT=""`
##### `--modem PRODUCTION|TEST` _changes what the console outputs_
##### `--show_isp [number]`
##### `--run_ussd [code]`
##### `--run_ussd_script <path to file>`
