### Deku Daemon
#### How to build and run
_Project's root dir_\
`make -B start`\
`./start --c <path to file> --mode TEST`
#### Start arguments
##### `--c [configuration file]`
* `DIR_ISP=""`
* `STD_NAME_REQUEST_FILE=""`
* `DIR_REQUEST_FILE=""`
* `DIR_SUCCESS=""`
* `DIR_SCRIPTS=""`
* `SOCKET_HOST=""`
* `SOCKET_PORT=""`
##### `--modem PRODUCTION|TEST` _changes what the console outputs_
##### `--show_isp [number]`
##### `--run_ussd [code]`
##### `--run_ussd_script <path to file>`
