Afkanerd-DekuDaemon  0.5
This program enables sending out bulk amount of SMS messages using locally available 2G/3G internet modems. It can also communicate across an SSH network to use other devices
Deku Daemon

Dependencies

  • g++ 4.8
  • make
  • doxygen - for generating documentations found in docs/
  • ModemManager - comes installed on most Linux systems
  • Linux system with Bash support

How to build and run

Project's root dir\ make -B start\ ./start --c <path to file> --mode TEST

Start arguments

<tt>--c [configuration file]</tt>

  • DIR_ISP=""
  • STD_NAME_REQUEST_FILE=""
  • DIR_REQUEST_FILE=""
  • DIR_SUCCESS=""
  • DIR_SCRIPTS=""
  • SOCKET_HOST=""
  • SOCKET_PORT=""

<tt>--modem PRODUCTION|TEST</tt> <em>changes what the console outputs</em>

<tt>--show_isp [number]</tt>

<tt>--run_ussd [code]</tt>

<tt>--run_ussd_script <path to file></tt>
